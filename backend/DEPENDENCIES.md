# Backend Dependencies

This file lists the packages required to run the backend app.

## Core runtime packages
- express
- dotenv
- @prisma/client
- prisma
- bcrypt
- jsonwebtoken
- helmet
- cors
- morgan
- zod

## TypeScript tooling
- typescript
- ts-node-dev

## Type declarations (Dev)
- @types/node
- @types/express
- @types/bcrypt
- @types/jsonwebtoken
- @types/helmet
- @types/cors
- @types/morgan

---

## Install commands

```bash
npm install express dotenv @prisma/client prisma bcrypt jsonwebtoken helmet cors morgan zod
npm install -D typescript ts-node-dev @types/node @types/express @types/bcrypt @types/jsonwebtoken @types/helmet @types/cors @types/morgan
```
